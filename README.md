# hr-agent

1. 业务价值与背景：

对公司HR来说内部工作起到降本增效的价值，且agent回复是基于人工手册及制度的更具有针对性的快速回答，可以通过agent检索到相关制度的内容块。

2. 项目的边界

该agent仅可以回复 基于  人工手册及制度  相关的内容，不想关的一律无法回复。

3. 重难点与风险
我觉得重难点方面在于语义检索和关键词检索，能否做到精准结合，让用户提出的问题能够得到精准的答案。
          风险在于，可能会出现幻觉事故，回复的答案不准确，让员工信以为真，造成经济合同风险；如果公司政策改变，Agent数据没有及时更新，也会出现问题。
4. 最核心模块设计

核心在于，语义检索及关键词检索；查询功能；数据库；重筛精筛；提示词工程；更新文本数据清洗模块
